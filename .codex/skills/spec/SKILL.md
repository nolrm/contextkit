Read `.contextkit/commands/spec/spec.md` and execute the spec pipeline workflow.

Single inline CTO pass per scope. Pass an overview `.md` file to start a new session (confirms before archiving existing progress). Run with no args to continue scope by scope. Use `--add` to append a new scope, `--extend` to add stories to an existing scope. Run `/loop /clear /spec` for overnight batch runs.
